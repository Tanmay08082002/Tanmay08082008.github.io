What I Did

Design and Analysis of Fire Fighting Robot

As a mechanical engineering student, I played a pivotal role in designing an autonomous firefighting robot. Our project aimed to create a robot capable of navigating buildings independently and extinguishing fires. I collaborated with a team to develop its structure, integrate advanced sensors, and implement sophisticated navigation algorithms. My contributions included 3D modeling, performance analysis, and system assessment using CAD software and simulation tools. This project enhanced my understanding of automation, fire safety, and robotics, while sharpening my skills in project management and system integration.

JARVIS Virtual Assistant integrated with a machine

In essence, this project is a virtual assistant that has been coupled with a very specific equipment (drilling, cutting, etc). The job of a virtual assistant is to launch the programme by just speaking the term that was introduced to the Python programme as a query. The whole set of queries and algorithms required to operate the machine efficiently are included in the Python programme.
